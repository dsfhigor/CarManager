# 🚗 Car Manager - Meu Veículo

O **Car Manager** é um assistente inteligente de manutenção veicular desenvolvido para Android. O aplicativo permite que motoristas gerenciem seus veículos, acompanhem a quilometragem e recebam recomendações preventivas baseadas no uso real, ajudando a evitar problemas mecânicos e aumentar a vida útil do carro.

## ✨ Funcionalidades

-   **Gerenciamento de Veículos:** Cadastro completo com modelo, ano, foto e quilometragem atual.
-   **Histórico de Manutenções:** Registro detalhado de trocas de óleo, filtros, pneus, freios e outros serviços.
-   **Análise Preditiva:** Identificação de possíveis problemas com base no tempo de uso e quilometragem acumulada.
-   **Recomendações Personalizadas:** Sugestões de manutenção específicas para o modelo do seu carro e manuais de fábrica.
-   **Personalização de Interface:** Suporte a Tema Claro, Escuro e Padrão do Sistema (persistido via DataStore).

## 🛠️ Tecnologias Utilizadas

-   **Linguagem:** [Kotlin](https://kotlinlang.org/)
-   **Interface:** [Jetpack Compose](https://developer.android.com/jetpack/compose) (UI Declarativa moderna)
-   **Design:** Material Design 3 (M3)
-   **Persistência de Preferências:** [Jetpack DataStore](https://developer.android.com/topic/libraries/architecture/datastore)
-   **Assincronismo:** Kotlin Coroutines & Flow
-   **Arquitetura:** MVVM (Model-View-ViewModel)
